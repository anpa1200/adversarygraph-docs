# Open Troubleshooting For An Error

**Level:** Simple

**Goal:** Move from a popup error to practical remediation.

## Real-Life Scenario

An enrichment action fails because an API key is missing, and the analyst needs a direct path from popup error to the relevant fix instructions.

## When To Use This

Use this workflow when you need a fast, low-friction action and want the output to remain traceable to evidence.

## Steps

1. Click Open troubleshooting in the error popup.
2. Follow the relevant fix and use Recheck to confirm resolution.


## Expected Result

Fast path from error to fix verification.

## Review Notes

- Keep source labels and evidence attached to every accepted result.
- Treat actor matches, enrichment hits, and matrix overlap as analytical signals until corroborated.
- Export only reviewed findings for customer, SOC, detection engineering, or executive use.
